# Análise de Dados com Python e Pandas

Este projeto realiza uma análise exploratória de dados utilizando Python, Pandas e NumPy.

## Objetivos

- Carregar e analisar um dataset
- Explorar estatísticas básicas
- Identificar padrões nos dados

## Tecnologias utilizadas

- Python
- Pandas
- NumPy

## Arquivos

- dataset.csv → base de dados
- analise.ipynb → análise dos dados
